FinWise

A Path to Canadian Financial Literacy

What it does

It helps Canadians build financial skills, all while maintaining engagement through personalization. The app provides several key features to achieve this:

  • A quiz assessment to check your current financial literacy level.
  • A personalized quiz based on your literacy level to help you learn.
  • A comprehensive glossary to learn more about financial and economic terms.

How We Built It

FinWise was built with a focus on simplicity and functionality:

  • Developed using HTML, CSS, and TypeScript.
  • Questions for the quizzes were gathered from various websites.
  • Leveraged Google Gemini to refine and clean up the codebase.
